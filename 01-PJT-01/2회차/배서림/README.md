# 프로젝트 01 - 파이썬 기반 데이터 활용
1. for 문

   ```python
   for 변수 in range(범위):
   ```

   

2. with 구문

   ```python
   with open('경로', '여는 방식', encoding = 'utf-8') as 이름:
   ```

   

3. 메서드

   ```python
   .read() # 파일 읽기
   .replace(' ', '') # 공백 지우기
   .split() # 문자열을 분해 후 배열을 반환
   .endswith() # ()로 끝난다면
   .append() # 리스트에 값을 더함
   .keys() # 딕셔너리의 키 값을 모아줌
   .items() # 딕셔너리의 키와 밸류를 동시에 얻음
   ```

   

4. 자료형

   ```python
   set() # 중복이 불가능함
   ```



5. 명령어

   ```python
   import 모듈 # 모듈 전체를 데려옴
   from 모듈 import 이름 # 모듈 내의 특정 이름만 데려옴
   ```

   

6. 함수

   ```python
   def 함수이름():
       수행문장
       return 반환값
   ```
