# 프로젝트 01 - 파이썬 기반 데이터 활용

## 후기
json파일을 처음으로 이용해봐서 신기했다. 솔직히 수업 들으면서도 json을 왜 사용하고 어떻게 쓴다는 건지 제대로 이해 못 했었는데 실습을 통해서 완전 이해해서 뿌듯했다!! 
파이썬을 4일동안 배우면서 거의 다 이해했다고 생각했는데도 역시나 막상 실습을 하니 바로바로 튀어나오지 않고 헷갈리는 개념들이 많았는데 실습이 역시 많이 도움이 됐다. 5번, 6번 문제를 풀면서 반복문에 딕셔너리, 리스트를 사용하는 방법들을 많이 이해해서 이제는 반복문을 사용하는 것이 두렵지 않아졌다.
실습시간에 4일동안 배운 내용을 모두 사용하고 복습할 수 있어서 좋았던 것 같다. 개별 프로젝트 말고 조별 프로젝트도 빨리 해보고 싶다!!!
